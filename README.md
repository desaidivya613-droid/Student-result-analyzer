# Student Result Analyzer (Python GUI)

## Description
This is a GUI-based application built using Python Tkinter to calculate student results.

## Features
- Input student marks
- Calculate average and grade
- Interactive buttons (Calculate, Clear, Exit)

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python
2. Open terminal
3. Run: python student_gui.py

## Learning Outcome
- Learned GUI development using Tkinter
- Implemented logic for grade calculation
- Built interactive user interface
